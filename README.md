# solar monitoring device

> monitors important metrics from solar panels and outputs to both the API as well as a physical display

## Metrics include:

1. Temperature
2. Light
3. Voltage
4. Current

## Key components:

1. Power
2. Rectifier
3. Regulator
4. Display
5. Micro Controller
6. Solar Panels (duh)
